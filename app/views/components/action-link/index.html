Example 1

Default

{{ govukPubActionLink({ heading: { text: "Go to NHS 111 online", href: "https://www.example.com" } }) }}

Example 2

With no wrapping text.

Uses a span and the govuk-pub-action-link__nowrap-text class.

{{ govukPubActionLink({ heading: { html: 'Coronavirus (COVID-19): what you need to do', href: "#" } }) }}
