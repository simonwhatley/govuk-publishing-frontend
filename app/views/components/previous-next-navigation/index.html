Components Previous and next navigation

Displays links that allow users to navigate between a series of pages or elements.

GOV.UK component guide

View the Nunjucks macro options on GitHub

Examples

Example 1

With title and text labels

{{ govukPubPreviousNextNavigation({ previous: { href: '#', title: { text: 'Previous page' }, label: { text: '1 of 3' } }, next: { href: '#', title: { text: 'Next page' }, label: { text: '3 of 3' } } }) }}

Example 2

No text labels

{{ govukPubPreviousNextNavigation({ previous: { href: '#', title: { text: 'Previous page' } }, next: { href: '#', title: { text: 'Next page' } } }) }}

Example 3

Only next

{{ govukPubPreviousNextNavigation({ next: { href: '#', title: { text: 'Next page' }, label: { text: 'Tax disc' } } }) }}
