Components Share links

Displays links for linking to or sharing the current page on social media.

GOV.UK component guide

View the Nunjucks macro options on GitHub

Examples

Example 1

{{ govukPubShareLinks({ items: [{ text: 'Facebook', icon: 'facebook', href: '#' },{ text: 'Twitter', icon: 'twitter', href: '#' }] }) }}

Example 2

{{ govukPubShareLinks({ items: [{ text: 'Share on Facebook', icon: 'facebook', href: '#' }] }) }}

Example 3

With departmental brand

{{ govukPubShareLinks({ brand: 'attorney-generals-office', items: [{ text: 'Follow the Attorney General on Twitter', icon: 'twitter', href: '#' }] }) }}

Example 4

Stacked icons

{{ govukPubShareLinks({ stacked: true, items: [{ text: 'Facebook', icon: 'facebook', href: '#' },{ text: 'Twitter', icon: 'twitter', href: '#' }] }) }}

Example 5

All available share links
