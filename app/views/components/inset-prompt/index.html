Components Inset prompt

A prompt to users represented as inset content.

GOV.UK component guide

View the Nunjucks macro options on GitHub

Examples

Example 1

Default

{{ govukPubInsetPrompt({ heading: { text: 'Please meet these requirements before publishing' }, description: { text: 'Document needs a summary before publishing (at least 10 characters)' } }) }}

Example 2

With items passed in as HTML

{%- set descriptionHtml %}
  • Document needs a title before publishing (at least 10 characters)
  • Document needs a summary before publishing (at least 10 characters)
{% endset -%} {{ govukPubInsetPrompt({ heading: { text: 'Please meet these requirements before publishing' }, description: { html: descriptionHtml } }) }}

Example 3

Error

{{ govukPubInsetPrompt({ error: true, heading: { text: 'There is a problem' }, description: { text: "‘Title here’ was not published" } }) }}

Example 4

List of items array

{{ govukPubInsetPrompt({ heading: { text: 'There is a problem' }, items: [{ text: 'Document needs a title before publishing (at least 10 characters)', href: '#' }, { text: 'Document needs a summary before publishing (at least 10 characters)', href: '#' }] }) }}
