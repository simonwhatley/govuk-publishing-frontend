Examples

Example 1

{{ govukPubFacetTags({ facets: [{ items: [{ label: { text: 'Aerospace and space' } }] }] }) }}

Example 2

With data attributes

{{ govukPubFacetTags({ facets: [{ items: [{ label: { text: 'Aerospace and space' }, button: { label: { text: '✕' }, attributes: { 'aria-label': 'Remove filter Aerospace and space', 'data-track-label': 'Aerospace and space', 'data-facet': 'sector_business_area', 'data-value': 'aerospace', 'data-name': '', 'data-module': 'remove-filter-link' } } }] }] }) }}

Example 3

{{ govukPubFacetTags({ facets: [{ heading: { text: 'Your business is in' }, items: [{ label: { text: 'Aerospace and space' } },{ label: { text: 'Automotive' } }] },{ heading: { text: 'You' }, items: [{ label: { text: 'Sell goods or provide services in the UK' } },{ label: { text: 'Import from the EU' } },{ label: { text: 'Export to the EU' } },{ label: { text: 'Provide services or do buisness in the EU' } }] },{ heading: { text: 'You employ' }, items: [{ label: { text: 'EU or EEA citizens' } }] },{ heading: { text: 'You exchange personal data by' }, items: [{ label: { text: 'Processing personal data from Europe' } },{ label: { text: 'Using websites or services hosted in Europe' } },{ label: { text: 'Providing digital services available to Europe' } }] }] }) }}

Example 4

Specifying a preposition, for example, ‘and’ (the default is ‘or’)

{{ govukPubFacetTags({ facets: [{ heading: { text: 'Your business is in' }, items: [{ label: { text: 'Creative industries' } },{ label: { text: 'Digital, technology and computer services' } },{ label: { text: 'Education' } }] },{ heading: { text: 'You' }, items: [{ label: { text: 'Sell goods or provide services in the UK' } }] },{ heading: { text: 'You employ' }, preposition: 'and', items: [{ label: { text: 'EU or EEA citizens' } },{ label: { text: 'Non EU or EEA citizens' } }] },{ heading: { text: 'You exchange personal data by' }, preposition: 'and', items: [{ label: { text: 'Processing personal data from Europe' } },{ label: { text: 'Using websites or services hosted in Europe' } },{ label: { text: 'Providing digital services available to Europe' } }] }] }) }}
