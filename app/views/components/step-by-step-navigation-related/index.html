Components Step by step navigation related

Links to step by step navigations that a page is part of.

GOV.UK component guide

View the Nunjucks macro options on GitHub

Examples

Example 1

{{ govukPubStepByStepNavigationRelated({ items: [{ text: 'Learn to drive a car: step by step', href: '#' }] }) }}

Example 2

If more than one link is passed to the component, they are output as a list.

{{ govukPubStepByStepNavigationRelated({ items: [{ text: 'Learn to drive a car: step by step', href: '#' },{ text: 'Learn to drive a motorbike: step by step', href: '#' }] }) }}

Example 3

The default title ‘Part of’ can be replaced if required.

{{ govukPubStepByStepNavigationRelated({ heading: { text: 'Also part of' }, items: [{ text: 'Learn to drive a car: step by step', href: '#' },{ text: 'Learn to drive a motorbike: step by step', href: '#' }] }) }}
