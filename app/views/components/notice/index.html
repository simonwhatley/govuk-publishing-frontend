Components Notice

Displays a notice to convey and highlight information.

View the GOV.UK component guide

View the Nunjucks macro options on GitHub

Examples

Example 1

{{ govukPubNotice({ heading: { text: 'Statistics release cancelled' } }) }}

Example 2

With a heading and description

{{ govukPubNotice({ heading: { text: 'Statistics release cancelled' }, description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' } }) }}

Example 3

With multiple paragraphs of description

{{ govukPubNotice({ heading: { text: 'Statistics release cancelled' }, description: { html: '

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

' } }) }}

Example 4

With data attributes

{{ govukPubNotice({ heading: { text: 'Statistics release cancelled' }, description: { text: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.' }, attributes: { 'aria-label': 'Notice', role: 'region' } }) }}

Example 5

With markup in the heading

{{ govukPubNotice({ heading: { html: 'Advisory Committee on Novel Foods and Processes has a separate website' } }) }}

Example 6

Without a heading

{{ govukPubNotice({ description: { text: 'Scheduled to publish at 8am on 25 April 2019' } }) }}
