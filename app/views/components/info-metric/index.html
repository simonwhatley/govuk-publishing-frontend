Example 6

{{ govukPubInfoMetric({ heading: { text: "Page views" }, description: { text: "Number of times the page was viewed" }, details: { summaryText: "About page views", html: '

This is a count of every time the page was viewed. For example, if someone visits page X, then goes to page Y and then back to page X again, page X will have 2 page views (and 1 unique page view).

Source: Google Analytics

' }, metric: { figure: "11,404,837" } }) }}

Example 7

{{ govukPubInfoMetric({ heading: { text: "Page views per visit" }, description: { text: "Number of times per visit the page was viewed" }, details: { summaryText: "About page views per visit", html: '

This figure is calculated by dividing Google Analytics page views by unique page views. It shows on average how many times a page was viewed during a user’s session. If a page has a high ratio (above 1.4), this indicates that users have to come back to that page within their session - investigate the navigation from that page further to identify any issues.

Source: Google Analytics

' }, metric: { figure: "1.2" } }) }}

Example 9

{{ govukPubInfoMetric({ heading: { text: "Number of feedback comments" }, description: { text: "Number of anonymous user responses to ‘Is there anything wrong with this page?’" }, details: { summaryText: "About number of feedback comments", html: '

You can use Feedback Explorer (‘Feedex’) to find out what users are saying about your content. A high rate of feedback could indicate a problem with the content.
