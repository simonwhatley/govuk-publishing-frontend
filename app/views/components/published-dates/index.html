Components Published dates

Displays dates to reflect when content was published and updated.

GOV.UK component guide

Examples

Example 1

Displayed above the content

{{ govukPubPublishedDates({ published: { timestamp: '1 January 1990' }, lastUpdated: { timestamp: '20 October 2016' } }) }}

Example 2

Displayed below the content

{{ govukPubPublishedDates({ attributes: { id: 'published-dates-toggle-2', lang: 'en' }, published: { text: 'Published', timestamp: '1 January 1990' }, lastUpdated: { text: 'Last updated', timestamp: '20 October 2016' }, history: [{ timestamp: '1 January 1990', text: 'First published' }, { timestamp: '20 July 1995', text: 'Updated to include information for 1994' }, { timestamp: '14 October 2000', text: 'Updated information on pupil premium reviews and what information schools need to publish on their websites.' }] }) }}

Example 3

Above and below content

{{ govukPubPublishedDates({ attributes: { id: 'published-dates-toggle-3a', lang: 'en' }, published: { text: 'Published', timestamp: '1 January 1990' }, lastUpdated: { text: 'Last updated', timestamp: '20 October 2016' }, linkToHistory: { href: '#published-dates-toggle-3b', text: 'see all updates' } }) }}
