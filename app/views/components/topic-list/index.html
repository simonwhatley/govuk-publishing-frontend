Examples

Example 1

Default

{{ govukPubTopicList({ items: [{ text: 'Environmental taxes, reliefs and schemes for businesses', href: '#' }, { text: 'Aggregates Levy: register or change your details', href: '#' }, { text: 'Pay environmental taxes', href: '#' }] }) }}

Example 2

With more link

{{ govukPubTopicList({ items: [{ text: 'Environmental taxes, reliefs and schemes for businesses', href: '#' }, { text: 'Aggregates Levy: register or change your details', href: '#' }, { text: 'Pay environmental taxes', href: '#' }], moreLink: { text: 'See more', href: '#' } }) }}

Example 3

With a brand colour, for example Department for the Environment, Food and Rural Affairs (department-for-environment-food-rural-affairs)

{{ govukPubTopicList({ brand: 'department-for-environment-food-rural-affairs', items: [{ text: 'Environmental taxes, reliefs and schemes for businesses', href: '#' }, { text: 'Aggregates Levy: register or change your details', href: '#' }, { text: 'Pay environmental taxes', href: '#' }] }) }}

Example 4

A version of the component with a smaller font size and reduced vertical spacing.

{{ govukPubTopicList({ small: true, items: [{ text: 'Environmental taxes, reliefs and schemes for businesses', href: '#' }, { text: 'Aggregates Levy: register or change your details', href: '#' }, { text: 'Pay environmental taxes', href: '#' }] }) }}

Example 5

With data attributes

{{ govukPubTopicList({ items: [{ text: 'Environmental taxes, reliefs and schemes for businesses', href: '#', attributes: { track_category: 'navSubtopicContentItemLinkClicked', track_action: '1.1', track_label: '/green-taxes-and-reliefs', track_options: "{dimension28: 2, dimension29: 'Environmental taxes, reliefs and schemes for businesses'}" } }, { text: 'Aggregates Levy: register or change your details', href: '#', attributes: { track_category: 'navSubtopicContentItemLinkClicked', track_action: '1.2', track_label: '/guidance/aggregates-levy-register-or-change-your-details', track_options: "{dimension28: 2, dimension29: 'Aggregates Levy: register or change your details'}" } }] }) }}
